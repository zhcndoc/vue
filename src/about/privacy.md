# Vue.js 隐私政策 {#vue.js-privacy-policy}

> 生效日期：2024年5月3日

本隐私政策描述了 Vue.js 组织（“Vue”、“我们”或“我们的”）在处理与你在本网站（https://vuejs.org）以及我们的开源相关网站（“网站”）上的信息时的做法，以及在本网站上向你提供的任何内容、相关文档、信息和服务（例如教程、支持开发者工作流的工具、对资源的访问等）（统称为“服务”）时的做法。本隐私政策描述了我们为支持我们的服务而处理的个人信息。

为清楚起见，本隐私政策不适用于以下任何情况：

1. 在 GitHub（https://github.com/）上提供的开源代码、文档或规范的使用，这些内容受适用开源许可证条款的约束；

2. 与参与 GitHub 上开源项目相关的拉取请求、问题以及任何其他交互或功能，这些内容受 GitHub 的条款和条件约束；或

3. 我们在 NPM（https://npmjs.com/）上发布的包的使用统计数据，这些数据受 NPM 的条款和条件约束；或

4. 由浏览器 / IDE 供应商收集的我们已发布的浏览器 / IDE 扩展的使用统计数据。此类统计数据受各供应商各自的条款和条件约束。

## 我们收集哪些类型的信息？ {#what-kinds-of-information-do-we-collect}

无论是通过我们的网站，还是通过我们发布的 npm 包或浏览器 / IDE 扩展，我们都**不会**收集或存储任何类型的个人数据。

我们可能会通过集成在我们网站中的第三方服务收集匿名化数据：

- **关于访问者访问我们网站的数据。** 我们的网站分析由 [Fathom Analytics](https://usefathom.com/) 提供支持，它不使用 cookie，并符合 GDPR、ePrivacy（包括 PECR）、COPPA 和 CCPA。使用这款注重隐私的网站分析软件时，你的 IP 地址只会被短暂处理，而我们（运行本网站的一方）无法识别你的身份。根据 CCPA，你的个人信息会被去标识化。你可以在 Fathom Analytics 的网站上阅读更多相关信息。

  - Fathom Analytics 的隐私政策：https://usefathom.com/legal/privacy

- **搜索功能的使用数据。** 我们的搜索功能由 [Algolia DocSearch](https://docsearch.algolia.com/) 提供支持，它不会进行任何类型的用户跟踪或指纹识别，也不使用 cookie。Algolia 服务符合 GDPR、CCPA，并通过 TRUSTe 认证。

  - Algolia 的隐私政策：https://www.algolia.com/policies/privacy/
  - Algolia 的安全与隐私合规：https://www.algolia.com/distributed-secure/security-compliance/

## 我们如何使用信息？ {#how-do-we-use-information}

收集上述数据的唯一目的是以尽可能注重隐私的方式了解我们网站的流量和使用情况，从而不断提升我们的网站和文档质量。根据 GDPR，其合法依据为“第 6(1)(f) 条；即我们在不断改进网站和业务方面具有合法利益。”如前所述，不会长期存储任何个人数据。

## 数据保留 {#data-retention}

所有收集的数据都存储在上述第三方服务上，并受这些服务各自的数据保留政策约束。

## 问题 {#questions}
