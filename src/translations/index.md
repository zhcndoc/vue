## 开始新的翻译 {#starting-a-new-translation}

Vue 文档最近经历了一次重大修订，因此其他语言的翻译仍然缺失或处于进行中。

我们欢迎社区为提供更多翻译所做的努力。翻译工作由 GitHub 组织 [vuejs-translations](https://github.com/vuejs-translations/) 负责管理。如果你有兴趣参与贡献，请查看 [翻译指南](https://github.com/vuejs-translations/guidelines/blob/main/README.md) 以开始上手。
